The ResPax Operator Network (RON) XML API is the window into a real-time tour operator portal network connecting in real-time to major Australian, New Zealand and Asia tour operators.

The ResPax Operator Network enables development of interfaces to directly interact with each operator retrieving tour details with imagery, check availability, check prices and commit bookings directly into the remote operators system.

The RON XMLRPC API is suitable for Travel Agents, Online Agents, Inbound Agents, Online Operators and Online Portals needing to develop a direct system-to-system interface between their system and the ResPax Operators.

This RON Test Tool has been provided to aid in exporting Operator and Tour content for initial uploads to Agent systems, and to test your XML code in a proven API environment with Error handling. If you have any questions regarding the use of this tool, please contact the ResPax Support Team.
